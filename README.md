# Contact Form Application

A modern and professional React.js contact form application for a Saudi Arabian company.

## Features

- Clean, elegant, and corporate design
- Responsive layout for mobile, tablet, and desktop
- Form validation
- Loading animation on submit
- Framer Motion animations
- Social media links
- Floating WhatsApp button

## Technologies Used

- React
- Tailwind CSS
- Framer Motion
- React Icons

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Build for Production

```
npm run build
```

## Components

- `ContactForm.jsx`: The main contact form with validation
- `ContactInfo.jsx`: Contact information and social media links
- `App.jsx`: Main application component

## Customization

- Replace the logo placeholder in `ContactForm.jsx` with your actual company logo
- Update contact information in `ContactInfo.jsx`
- Modify colors in the Tailwind classes to match your brand
